## ?? TIPS:

### Custom Domain (Optional)
Jika mau ganti link dari `random-name-123.netlify.app` jadi nama sendiri:
1. Klik **"Site settings"**
2. Klik **"Change site name"**
3. Ketik: `instrategic-market` (atau nama lain)
4. Link jadi: `https://instrategic-market.netlify.app`

### Jika Mau Update Website Nanti
1. Download ZIP lagi dari GitHub
2. Buka: https://app.netlify.com
3. Login (kalau belum)
4. Klik site Anda
5. Drag & drop folder baru
6. Website otomatis update!

## ?? TROUBLESHOOTING:

### Masalah 1: "You need to sign in"

**Solusi:**
- Netlify minta login (gratis!)
- Klik **"Sign up"**
- Pilih **"Email"** atau **"GitHub"**
- Ikuti instruksi
- Setelah login, ulangi drag & drop

### Masalah 2: Chart tidak muncul

**Solusi:**
- Hard refresh: Ctrl + Shift + R
- Tunggu 1 menit (first load agak lama)
- Buka mode incognito
- Jika masih kosong, kasih tau saya!

### Masalah 3: Error saat upload

**Solusi:**
- Pastikan extract ZIP dulu (jangan drag file .zip)
- Drag folder yang ada file index.html di dalamnya
- Coba browser lain (Chrome lebih stabil)
