# InStrategic - Market Overview Clone

This is a complete clone of the InStrategic Market Overview website from https://invest-genius-25.preview.emergentagent.com/market

## Files Included

- **index.html** - The main HTML file with all styles and content
- **logo.png** - The InStrategic logo image

## Features

? Complete visual clone with all original styling
? Fully responsive design using Tailwind CSS
? Glass morphism effects (glassmorphism cards)
? Gradient backgrounds and purple theme
? Market data display:
  - IHSG (Jakarta Composite Index) with real-time data
  - Top Gainers section with 5 stocks
  - Top Losers section with 5 stocks
  - Sector Performance grid with 6 sectors
? Navigation bar with links to different sections
? Hover effects and transitions
? Mobile-responsive menu button

## How to Use

Simply open `index.html` in any modern web browser:

```bash
open index.html
```

Or double-click the file in your file manager.

## Technical Details

- **Framework**: Static HTML with embedded CSS
- **CSS Framework**: Tailwind CSS (extracted from original)
- **Design**: Dark theme with purple accents (#6800FC)
- **Effects**: Glass morphism, gradients, hover animations
- **Typography**: System fonts (Apple system, Segoe UI, etc.)

## Browser Compatibility

Works best in modern browsers:
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+

## Customization

To customize the content:
1. Open `index.html` in a text editor
2. Find the relevant section you want to modify
3. Update the text, numbers, or styling
4. Save and refresh in your browser

## Notes

- All styles are embedded in the HTML file for easy portability
- The logo image must be in the same directory as index.html
- Links currently point to the original routes (/, /market, /stocks, etc.)
- This is a static clone - no backend functionality or real-time data updates
